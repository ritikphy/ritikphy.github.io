# Ritik Dubey — Academic Homepage

A minimalist academic homepage for theoretical physics.

## Where to put your files

- Hero images: `images/hero/`
- Profile photos: `images/profile/`
- Quotes: `data/quotes.json`

Use filenames like:
- `hero01.jpg`, `hero02.jpg`, ...
- `profile01.jpg`, `profile02.jpg`, ...

The site picks one hero image and one quote randomly on each visit.
